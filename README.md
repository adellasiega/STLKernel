# STLKernel

A Python implementation of Kernel Methods for **Signal Temporal Logic (STL)** on Stochastic Processes, integrated with the [TorCheck](https://github.com/ailab-units/TorCheck) library.

## Project Structure
- `src/`: Core logic including `STLKernel`
- `kernels/`: Specific kernel functions (Gaussian, k0, k').
- `notebooks/`: Usage examples and model checking benchmarks.
